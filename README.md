# Zepto SQL Data Analysis Project

## Overview
This project analyzes Zepto inventory data using SQL to derive insights on product pricing, discounts, and inventory management.

## Tools Used
- PostgreSQL
- SQL
- pgAdmin

## Dataset
The dataset contains 3700+ product records including product name, category, MRP, discounted price, stock availability, and product weight.

## Business Questions Solved
1. Top 10 best-value products based on discount percentage
2. High MRP products that are out of stock
3. Estimated revenue by product category
4. Product categorization by weight
